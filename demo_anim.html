CopilotKit

构建深度集成的 AI 助手和 Agent

CopilotKit 通过 useCopilotAction 和 useCopilotReadable 两个核心 API，让 React 应用原生集成 AI 能力。Agent 可以直接操作前端 UI 状态，实现真正的「人机协同」。

核心功能

三大支柱：Readable · Action · Chat

📖

useCopilotReadable

实时暴露应用状态给 AI，让 Agent「看懂」你的前端。

⚡

useCopilotAction

注册自定义函数，AI Agent 可直接调用前端方法执行操作。

💬

CopilotChat

可嵌入的 AI 聊天组件，支持上下文感知和流式响应。

🔗

CoAgent

基于 LangGraph 的 Agent 协作框架，支持多 Agent 协同。

CopilotKit 让你的应用「理解」用户意图，并「主动」执行操作。从简单的文本补全到复杂的多步骤工作流，都可以通过声明式 API 实现。

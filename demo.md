## 🛠️ 推荐工具链

| 场景 | 工具 | 效果 |
|------|------|------|
| 写作 | Claude / ChatGPT | 初稿 → 润色 |
| 编程 | Cursor / Copilot | 代码补全 + Review |
| 设计 | Midjourney / DALL-E | 快速出图 |
| 数据 | Code Interpreter | 自动分析 |
| 演示 | **Slide Tool** | Markdown → PPT |

🎤 给大伙儿推荐几个贼好用的工具。写作用Claude，编程用Cursor，设计用Midjourney，数据分析用Code Interpreter。咱这个Slide Tool更厉害，Markdown直接变PPT，老省事儿了！

## ⚠️ 几个踩坑经验

- ❌ 不要完全信任 AI 输出，**一定要验证**

+++

- ❌ 不要什么都丢给 AI，**判断力是你的核心价值**

+++

- ❌ 不要忽略隐私和安全，**敏感数据别上传**

+++

- ✅ 建立自己的 Prompt 模板库，**复用 > 重复**

🎤 但是啊，有几个坑你可得注意。AI输出的东西一定要验证，别啥都信。判断力才是你的核心价值。敏感数据千万别往上传。还有就是建个Prompt模板库，复用比重复强多了！
